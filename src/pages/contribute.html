Community

Contribute

Waterfall is young; clear issues, docs PRs, and CLI polish are all welcome.

Ways to help

  • Improve this site: templates under src/, build with npm run build.
  • Harden the CLI: integration tests, backends, and prompt copy in waterfall-cli.
  • Sharpen the spec conventions in waterfall-spec so agents and humans stay aligned.

Workflow

Use short-lived branches, run tests for the package you touch, and keep commits scoped. Match existing style and naming in each repo.
