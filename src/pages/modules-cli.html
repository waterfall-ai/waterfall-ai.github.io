Waterfall CLI

Run spec workflows from the terminal: branch-aware commands, dry-run plans, and real agent invocation when you choose.

What it does

  • Resolve your spec root via flags, env, .waterfall, or waterfall.json.
  • CR lifecycle: cr start, stepwise edges (cr to rq, …), cr all, cr finish.
  • Subsystem and horizontal scaffolds: sys start, horizontal create / horizontal update.
  • Backends: cline-ollama, Cursor, Claude Code—with embed or reference prompt delivery.
